# Report exports

Reports are stored as validated TipTap JSON and exported from authenticated server routes after project and report ownership checks.

Supported formatting: paragraphs, headings, bold, italic, links, bullet/ordered lists, block quotes, code blocks, tables, and hard breaks. Links are emitted only when they use HTTP or HTTPS. HTML output is generated by escaping text and rendering only supported nodes/marks into a standalone UTF-8 document with print-friendly CSS. Markdown is UTF-8 text. PDF uses `@react-pdf/renderer` in a Node.js route and includes report title, metadata, wrapped text, and page numbers.

Limitations: unsupported TipTap extensions are ignored during export; complex table styling is simplified; PDF output prioritizes reliable readable text over exact browser-editor visual fidelity.
